Project 3.1 – The HTML Poster


Self-portrat illustration by Robert M. Kerr. Rendered in HTML and CSS.

• Choose a portrait you took which focuses on the face.
• This can be a self-portrait, a photo of someone else, or a photo of a pet.
• In Illustrator, create two drawings of the portrait using geometric shapes. Grayscale only.
• Start with an artboard that is sized to your screen. Change Illustrator preferences to work with pixels.
• The drawing should be minimal and convey emotion (ex. Apple Animojis).
• Consider the types of shapes you can create in HTML – rectangles, circles, ovals and triangles.
• Once your drawing is approved, recreate it using only DIVs in HTML.
• Use DIVs position absolute, and start with the drawing aligning to the top and left of your browser.

Project 2.2 – Web-Based poster – a focus on the scroll


• Create a drawing of how you will distribute the information vertically.
• Create a detailed drawing of your project in Illustrator.
• Determine percentage breakpoints, with the total adding up to 100%.
• This page should feel like a scrollable poster, and scale with the browser.

Project 2.1 – Information Architecture and a Web-Based poster – Styling the Dieter Rams Essay


• First, style your essay with a focus on legibility and organization of information.
• Focus on typography and spacing between paragraphs.
• The final HTML page should be: legible, clear, easy to read.

• Next, draw a wireframe for a second version of the page that focuses on rhythm and vertical movement. This page should feel like a scrollable poster.

Project 1.2 – HTML Calculator – Based on the work of Dieter Rams


• We will first research the work of Dieter Rams.
• Create a moodboard of elements from his designs to use for the calculator.
• Design your calculator in Illustrator. Take into account: size in browser, alignment, color, typography. Include a roll-over state on one of the buttons.
• Code your design using your previous iteration and update CSS to reflect your new design.
• Use the W3 Schools CSS tutorial to determine the styling: CSS Tutorial on W3 Schools

Project 1 – HTML Calculator


We will code a simple calculator using HTML and Javascript in class. We will then add CSS code to style the calculator. The calculator will be styled using CSS with the following considerations:
• Functionality – how does it work?
• User experience – how will a user interact with it?
• User Familiarity – how do calculators usually look?
• Aesthetics – does the style add or detract from the function?
