Project 1 – HTML Calculator


We will code a simple calculator using HTML and Javascript in class. We will then add CSS code to style the calculator. The calculator will be styled using CSS with the following considerations:

• Functionality – how does it work?
• User experience – how will a user interact with it?
• User Familiarity – how do calculators usually look?
• Aesthetics – does the style add or detract from the function?
